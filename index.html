About the Game

ROAM follows Nuggets – a small, feathered Sinosauropteryx prima – on a heartfelt journey to find its elusive mate’s nest. Watch as Nuggets travels through a beautifully stylised Cretaceous birch forest, painted in hues of gold and amber.

Help Nuggets explore an environment where they will jump, crouch and weave through dense forest, feast on fruits and insects and encounter other dinosaurs along the way

I worked as the main Artist and Game Designer on this project - I handled the 3D modeling, rigging and texturing of characters, did the 2D artwork including the Key art and programmed the interactions in the game in Unity using C#

About the project

Technical art project in Unity - the project is a 3D rendition of artworks by Blanche McManus for Alice In Wonderland. This project explored using Shader Graph in Unity and getting a stylized look that mimics the works of Blanche McManus.
I used Shader graph to create the flowing tea, wind blowing table-cloth, floating paintings and the caterpillar's pipe - as well as the custom lighting used to apply the pencil shading texture to achieve a similar look to the original.

About the Project

This project saw the use of Unreal Engine, ZBrush and Adobe Substance Painter to create the models and environment

ENIGMA CITY

About the project

Procedural city generation project using Geometry Nodes in Blender. The project focuses on creating modular assets that can be reused to create a sprawling cityscape. This resulted in a city that could be randomized and changed easily whilst providing large amounts of detail.
The main city is randomly generated with a few modular pieces and renders in real-time with Blender's EEVEE
